Another Evolution Update - V11.0

New Features

  • Evogoals: Replacing the old endgoal, every evolution now has a second goal. Completing it takes time but grants permanent rewards. Evolving isn’t always the best choice anymore!
  • Completing an evogoal grants a +2.5% auto and click multiplier.
  • Extra evogoals added to evolutions: 1–5, 9, 14, 19, 24 (more coming soon).
  • New evolution requirement: Buy a minimum number of upgrades during each evolution.
  • MEGA Upgrades: Now available for auto-upgrades too. Clicking and autopower share the same count — choose wisely!
  • New sound alert: Pop-ups now play a notification sound.
  • Settings screen: Access reset and sound toggle options through the new settings button.

Balance Changes

  • MEGA-upgrades no longer count towards the 10 required upgrades.
  • Adjusted some evolution requirements.
  • Removed the old endgoal.
  • First coin batch now awarded at 150 clicks instead of 350.
  • Added a new evolution requirement: buy at least 5 × your evolution level upgrades (clicker or auto).

UI/UX Improvements & Quality of Life

  • Welcome message and update logs are now combined in a fixed position for consistent display.
  • The Evolve button turns green at max level (25).
  • Evolution messages now clearly state what you lose on evolving.
  • Second page of the Coin Shop is accessible (content coming soon).
  • Achievement rewards use the improved pop-up UI.
  • MEGA-upgrade effect duration shortened.
  • Warning shown if you haven’t reset after an update (ignore if you already did).
  • Game loads directly into the regular shop after loading a save.
  • Some buttons moved due to the new settings screen.
  • Changed tho look of the changelog

Bug Fixes

  • Fixed power/autopower calculation issues after evolving.
  • Offline autoclicker and coin autoclicker now appear consistently on desktop and mobile.
  • Prices and offline multipliers are visible again.
  • Minor typo and spelling fixes.
  • Rank shop items unlock at evolution 8 instead of 9.
  • Reduced permaupgrade font size for better fit on smaller screens.
  • Clicker/offliner free upgrades start at evolution 3 instead of 4.
  • Fixed autoclicker UI update issue when coin autoclicker not purchased.

Known Issues

These minor bugs are not urgent and may take some time to fix.

  • Since v6.0: Autosaving during specialization selection may cause save lockout. Temporary fix: autosave disabled on this screen, forced save before Evolution 3.
  • Since v5.0: On mobile, upgrade button colors only update after tapping elsewhere.
  • Multiple simultaneous pop-ups overwrite timers, causing them to disappear too quickly.

  • Improved number formatting for prices and stats.
