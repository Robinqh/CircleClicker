Welcome to Circle Clicker
Version 1.0

Circle Clicker is a browser-based clicker game in which the goal is to reach the 35th evolution.
Throughout the game, players will unlock upgrades and buffs that'll help you reach the end goal.

Each evolution features three goals:
- Goal 1: changes every evolution.
- Goal 2: maintains a consistent pattern but becomes progressively more difficult.
Completion of both goals is mandatory to evolve.
- Goal 3: optional and more challenging, offering a small permanent bonus if completed.

At Evolution 3, players select one of four specialities, each providing a unique playstyle and strategic focus, allowing for diverse approaches to progression.

The game also features three distinct shops:
- Base Shop: provides standard upgrades for progression.
- Coin Shop: improves the effectiveness of Base Shop upgrades.
- Rank Shop: offers offline upgrades and various miscellaneous items.

Key features include:

  • Offline progression, enabling points accumulation while away from the game.
  • Mystery gifts providing good, or bad, things.
  • Achievements, adding additional objectives.
  • Hidden easter eggs throughout the game.
  • A settings screen to adjust gameplay preferences.
  • Endgame evolutions (25–34) that present a higher difficulty than before.
Enjoy!
