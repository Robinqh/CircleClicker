Version - 11.21

Balance Changes

  • EvoLoss multiplier bought from the CoinShop now reduced EvoLoss by >-5% instead of -10%
  • Evo-17 and Evo-19 now give a -5% to the price multiplier
  • You now need to buy at least 3 upgrades before performing your first evolution

UI/UX Improvements, Quality of Life Changes, and Small Tweaks

  • Updated changelog layout for a cleaner look
  • Removed the warning message when closing the changelog without resetting; the reset button warning is sufficient
  • Added an animation when opening the changelog
  • Updated the evolve warning message
  • Updated the evolve rewards message
  • You now choose your speciality after reaching Evo-3 instead of during evolution; the forced save is also removed
  • You can now enter the second (currently empty) Rank Shop page! Content coming soon
  • Improved the appearance of the progress bar

Bug / Issue Fixes

  • Changelogs no longer appear in OG-Mode
  • You will never lock yourself out of evolving if something goes wrong while choosing a speciality
  • The Toggle Music button is now visible in the settings screen (mobile)
  • The second (empty) page of the Coin Shop is now accessible
  • The Mystery Gift popup now disappears after 2.5 seconds
  • Fixed various issues with the evo progress bar
  • Fixed some mobile UI issues
  • (V11.21): Fixed some small issues in the changelog
