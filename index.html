Update Version 10.0
THE BIGGEST UPDATE YET
August 1, 2025 (20:00)

Last large update (V9.0): July 31, 2025 (22:30)

This update:

New Features / Reworks

  • Evolution Hard Cap: You can now have a maximum of 25 evolutions! (To be increased to 50 in a future update!)
  • Free Evolutions: Evolutions no longer cost anything — but you must now complete a challenge before each one! This also means you can sometimes evolve multiple times in a row if you’re fast enough.
  • Dynamic Requirements: Evolution goals now display how far you’ve progressed, not just the final requirement.
  • Evolver Spec Rework: Evolver spec can no longer skip evolution requirements. However, their evo loss percentages are much improved as compensation.
  • New Easter Egg: A secret has been added… can you find it?
  • Endgoal Update: The final goal is now 50 million Points (was 10 million) — but it’s now based on all-time Points instead of current Points.
  • MEGA Upgrades: Every 10th clicker upgrade now results in a MEGA upgrade — granting 5× the clicking power of a normal upgrade.
  • Secret V0.1 Mode: A hidden button in the UI brings you back to the original version of the game (V0.1). All code is faithfully copied; only a warning and return button are added. Good luck finding it!

Balance Changes

  • EvoLoss Update: Losing an evolution now always resets Power and AutoPower — even if it previously didn’t.
  • Clicker Spec Buff: Clicker spec now starts every evolution with 10 free Clicking Power upgrades (up from 1).
  • Offliner Spec Buff: Offliner spec now starts every evolution with 10 free AutoPower upgrades (up from 0).
  • Offline Nerf: The offline multiplier now starts at 10% instead of 25%, reducing the maximum possible bonus.
  • Hard Cap: You can now only evolve up to evolution 25 — for now.
  • RP → Points Trade: RankPoints now convert to Points at a 1% rate, but with no upper limit.
  • Mystery Gift Change: The Free Evolution reward has been replaced with a 2× Points reward.
  • Rebalanced Evo Requirements: Most old requirements are now more difficult, and several new ones were added.
  • Evolution Rewards Updated: Several rewards tied to evolutions have been rebalanced or improved.
  • Spec Rewards Improved: Specialization benefits have been buffed overall.
  • Rank Rewards Tweaked: Rank bonuses have been rearranged and rebalanced.

UI/UX – Quality of Life – Tweaks

  • Coin Display: You can now see your current Coins directly inside the Coin Shop!
  • Stat Panel: The Stats page has been visually updated for clarity.
  • PermaPower Tooltip: PermaPower now includes an info box explaining its effect.
  • Small UI Tweaks: Various minor adjustments to layout and visuals (especially on desktop).
  • Updated Endgoal Message: The message shown after reaching the endgoal has been improved.
  • Spec Explanations: Specializations now provide clearer descriptions of what they do.
  • Locked Buttons Hidden: Buttons for content you haven’t unlocked are now hidden instead of greyed out.
  • New Circle Color: The main click-circle now better matches the overall aesthetic of the game.
  • Offliner Rank Shop Access: The Offliner spec can now enter the Rank Shop at Evolution 3. However, all non-automatic upgrades remain locked until Evolution 8.

Bug & Issue Fixes

  • Mobile Click Bug Fixed: Tapping the Circle no longer shifts the layout on mobile devices.
  • Color Reset Fix: Color themes now update immediately after a reset.
  • Click Sound Fix: Fixed a bug where clicking too fast caused audio issues.

Code/Dev Optimizations

  • Minor clean-up for the HTML document.

Previous large update (V9.0) - Stabalizing The Game

New Features

  • Pop-ups! All messages now appear as sleek pop-ups — more readable, less intrusive, and visually improved!
  • Sound! Clicking the circle now plays a short sound effect. More sounds are planned for future updates!
  • Toggle Sound: You can now mute or unmute the game using the new “Toggle Sound” button!
  • Shop Highlighting: The currently active shop button is now golden instead of green. Enjoy the new aesthetic — and a special message while browsing certain shops!

Balance Changes

  • The first evolution now costs 500 instead of 750, making early progression a bit smoother.
  • All "Click X times" achievement rewards have been multiplied by 10× to better match the new coin system (still around 33% lower than they originally were).
  • The Clicker specialization now gains an extra +5% click multiplier at Evolution 4 to compensate for already having access to the Coin Shop.
  • Evolution requirements have been completely rebalanced — most old ones are now harder, and many new ones have been added.

UI/UX – Quality of Life – Tweaks

  • Dynamic Requirements: Evolution goals now show your progress instead of just the final amount required.
  • Rank Glow: All rank names now glow — with an especially impressive glow for the mysterious [REDACTED] rank.
  • Locked Items: Things you haven’t unlocked yet are now purple-ish, instead of plain black.
  • Improved Autosave Logic: The game only saves on first load if autosave was already enabled.
  • Visual Polish: Sidebars have clearer borders (on desktop), and several background colors have been updated.
  • Readded Effect: The circle now grows/shrinks when clicked again!

Bug & Issue Fixes

  • Buyer spec's bonus RPpS at certain evolutions now applies correctly.
  • The evolve button now turns orange after resetting if requirements aren’t met (instead of staying black).
  • Buying an auto upgrade no longer immediately triggers an autoclick.
  • The autoclicker stat panel now correctly shows the autoclicker multiplier.
  • Total coins are now in sync with the new 15× coin system, making coin achievements more achievable.
  • After resetting, players are now correctly sent back to the Base Shop.
  • Updated the "Coming Soon" section.

Code Optimizations

Known Bugs

(These bugs are minor and not urgent, so they may take some time to fix.)

  • (Since always) If multiple pop-ups of the same type trigger at once, their timers overwrite each other — causing them to disappear too quickly.
  • (Since ???) Autoclickers do not update the evolution progress requirement until you click manually.
